# Blog YouTalk

Адаптивная верстка блога и страницы статьи

## Страницы

- `index.html` - страница блога
- `article.html` - страница статьи

## Используемые технологии

- HTML5
- CSS3
- CSS Grid
- Flexbox
- БЭМ

## Структура проекта

- `index.html` - главная страница
- `article.html` - страница статьи
- `styles/main.css` - основной файл стилей
- `images/` - папка с изображениями
- `fonts/` - папка со шрифтами

## Как открыть

Откройте `index.html` или `article.html` в браузере
